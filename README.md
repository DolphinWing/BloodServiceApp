快速查詢最近的捐血活動，以及台灣各血液中心目前的庫存狀態。

資料來源為[醫療財團法人台灣血液基金會](http://www.blood.org.tw/)

本程式需要網路連線來抓取最新資料，以便提供最即時資訊。
